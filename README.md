# Socket

## Installation
Clone [git](https://github.com/Basil-jiji/Socket.git) repository 

To make it executable
```bash
chmod +x s.py
```
To Run
```bash
./s.py
```
## Usage
Download and install [netcat](https://nmap.org/ncat/)

At the terminal,
```bash
nc -nvlp 7777
```
Now Open another terminal
```bash
./s.py
```
